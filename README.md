# Chameera Visuals Portfolio

A high-end, cinematic photography portfolio website designed with a dark luxury aesthetic and "antigravity" floating effects.

## Setup Instructions

Since this is a static HTML/CSS/JS website, you don't need to install any complex dependencies like Node.js.

1.  **Open the Website**:
    Simply double-click `index.html` to open it in your default web browser.

2.  **Customization**:
    -   **Images**: Replace the placeholder images in the `images` folder (or update the URLs in `index.html` and `style.css`) with your own high-resolution photography.
    -   **Text**: Edit `index.html` to update your bio, services, and contact details.
    -   **Colors**: You can tweak the gold/blue accent colors in `style.css` under the `:root` variables.

## Features

-   **Cinematic Hero Section**: Full-screen intro with smooth zoom animation.
-   **Antigravity Effects**: Floating elements and a magnetic custom cursor.
-   **Portfolio Gallery**: Filterable grid for Portraits, Events, and Commercial work.
-   **Responsive Design**: Fully optimized for mobile and desktop.
-   **Smooth Scrolling**: Enhanced navigation experience.

## Technologies Used

-   HTML5
-   CSS3 (Variables, Flexbox, Grid, Animations)
-   JavaScript (ES6+)
-   Font Awesome (Icons)
-   Google Fonts (Outfit & Playfair Display)
